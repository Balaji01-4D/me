I'm working on a modern, interactive CLI client for PostgreSQL to make querying a bit nicer. 

- Added word-based autocompletion, syntax highlighting, and a persistent command history folder.
- Working on supporting common Postgres backslash commands (like \d, \l) and planning to add context-aware suggestions soon.
