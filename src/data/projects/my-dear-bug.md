A simple platform I built to let developers share their wildest bug confessions.

- Wrote the backend in Go using Gin and GORM, keeping the architecture pretty simple and clean.
- Added basic features like pagination, tag filtering, and a trending algorithm for posts.
- Dealt with IP-based upvote limits to prevent spam, a rate limiter, and admin-only routes for moderation.
