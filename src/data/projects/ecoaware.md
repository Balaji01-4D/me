A backend solution for tracking issues and operations.

- Built a secure REST API with proper role-based access control (RBAC) and JWT authentication using Spring Security.
- Added features for secure image uploads, tracking issue statuses, and basic CRUD stuff for different user roles.
- Kept the architecture clean and straightforward, mapping it all directly to the PostgreSQL database schemas.
