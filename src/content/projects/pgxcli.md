Developing a modern interactive PostgreSQL CLI client with a customizable REPL experience. 

- Implemented word-based autocompletion, syntax highlighting, and persistent smart command history.
- Added support for common PostgreSQL backslash commands (e.g., \d, \l) with plans for context-aware suggestions.
